How to use this web application?

Generally,numbered Jenga has 48 tiles, numbered from 1 to 48. On clicking the "Roll the Dice!" button, this application will provide you a random number that lies within the initial range, and then it will add this number to the "Top 2 rows" column. The top 2 rows are not considered for the next draw.

When you encounter a situation where there are only 1 or 2 tiles left in a row which cannot be moved, you can click on that number to mark it as prohibited. Have fun palaying Jenga!!
